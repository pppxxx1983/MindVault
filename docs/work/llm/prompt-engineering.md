# 提示工程

系统性整理 Prompt 设计模式与技巧。

---

## 核心原则

1. **明确指令** — 清晰表达你想要什么
2. **提供上下文** — 给予足够的背景信息
3. **分步思考** — 复杂任务拆分为步骤
4. **示例驱动** — 用 few-shot 示例引导输出格式

## 常用模式

### Chain of Thought（思维链）

```
请一步步分析这个问题：
1. 首先理解问题的核心
2. 然后列出关键因素
3. 最后给出结论
```

### Role Play（角色扮演）

```
你是一位资深的游戏引擎开发者，拥有 10 年的 Python 开发经验。
请帮我 review 以下代码...
```

### Format Specification（格式指定）

```
请以 JSON 格式输出，包含以下字段：
- name: 字符串
- score: 数字 0-100
- reason: 字符串
```
