## 微调方法对比

| 方法 | 数据需求 | 效果 | 成本 | 适用场景 |
|------|----------|------|------|----------|
| Full Fine-tuning | 大量 | 最佳 | 高 | 有充足数据和算力 |
| LoRA | 少量 | 接近全量 | 低 | 资源有限 |
| QLoRA | 少量 | 略低于 LoRA | 最低 | 显存极度受限 |
| Prompt Tuning | 极少 | 一般 | 最低 | 快速适配 |

## 微调 Checklist

- [ ] 确定目标任务（分类、生成、对话等）
- [ ] 准备高质量数据集（清洗、格式化）
- [ ] 选择基座模型
- [ ] 选择微调方法（LoRA / QLoRA / Full）
- [ ] 配置训练超参数
- [ ] 监控训练指标（loss、eval loss）
- [ ] 评估模型效果
- [ ] 导出和部署
