# LLM Wiki

大语言模型知识库 - 系统性整理 LLM 相关知识、工具和实践。

## 目录结构

- `models/` - 主流模型介绍和对比
  - 各厂商模型能力、价格、上下文长度等
- `tools/` - LLM 工具和框架
  - 开发框架、部署工具、评估工具等
- `prompts/` - Prompt 模板和技巧
  - 常用 Prompt 模式、最佳实践
- `tutorials/` - 教程和指南
  - 入门教程、进阶技巧、实战案例
- `resources/` - 学习资源和链接
  - 论文、博客、视频、社区
- `papers/` - 重要论文整理
  - 经典论文解读和笔记
